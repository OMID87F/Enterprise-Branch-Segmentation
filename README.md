# Enterprise Branch Segmentation
>**Status**: 🚧 Work in Progress!

## Overview
This project focuses on designing and implementing a small enterprise network with an emphasis on network segmentation, security, and operational best practices.

Unlike my previous enterprise lab, this project is being developed incrementally with greater attention to security hardening, documentation quality, validation, and troubleshooting.

The goal is to build a realistic enterprise environment that can continue evolving as new networking concepts are learned.


## Objectives
* Implement VLAN-based network segmentation.

* Configure Inter-VLAN Routing.

* Deploy centralized DHCP services.

* Apply firewall policies between network segments.

* Harden network management services.

* Implement basic Quality of Service (QoS).

* Integrate centralized Syslog logging.

* Practice troubleshooting through the implement.
